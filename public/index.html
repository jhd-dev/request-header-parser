Request Header Parser

This is an API to find the IP address, language, and operating system from a browser.

Example Usage:

$.getJSON("https://porygonj-request-header-parser.herokuapp.com/whoami", function(data){
    console.log(data); // {"ipaddress":"50.195.16.177","language":"en-US","software":"Windows NT 6.1; Win64; x64"}
});
